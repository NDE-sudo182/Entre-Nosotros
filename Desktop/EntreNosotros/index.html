Esta página no fue creada solo para mostrar fotos o escribir frases bonitas. Fue creada para recordar. Para capturar la esencia de una etapa que marcó nuestras vidas. Aquí queda guardado el reflejo de nuestra trayectoria: los momentos que nos moldearon, los lazos que formamos, y cómo, entre risas, tropiezos y aprendizajes, cada uno ayudó a construir al otro.
